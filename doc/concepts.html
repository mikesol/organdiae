Organdiae in Six Concepts¶

The following presents six concepts [1] around which Organdiae is designed. The order of presentation is mostly to create cliffhangers between the sections, although there is probably some hierarchical logic at play too.

  • Classes/Vertices and Instances/Graphs
  • Filtering graph growth
  • Edges Part I (Features and Bugs)
  • Edges Part II (Getting Edgier)
  • Organdiae is meant to be used interactively
  • Bootstrapping and inheritance: Back to meta-classes
  • Conclusion

Before reading these documents, if you are not familiar with Python, read this first. Organdiae will be of little practical value to you if you do not feel comfortable in the Python interpreter.
